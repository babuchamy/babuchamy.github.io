Ready-to-Use Oracle Scripts

Copy, paste, and run. Battle-tested scripts for monitoring, troubleshooting, and automation.

active_sessions.sql 
SELECT s.sid, s.serial#,
  s.username, s.status,
  s.machine, q.sql_text
FROM v$session s
JOIN v$sql q
  ON s.sql_id = q.sql_id
WHERE s.status = 'ACTIVE'
  AND s.type != 'BACKGROUND';
Find Active Sessions
tablespace_usage.sql 
SELECT tablespace_name,
  ROUND(used_space*8192/1024/1024) used_mb,
  ROUND(tablespace_size*8192/1024/1024) total_mb,
  ROUND(used_percent,2) pct_used
FROM dba_tablespace_usage_metrics
ORDER BY pct_used DESC;
Tablespace Usage
blocking_sessions.sql 
-- Find blocking sessions
SELECT l1.sid blocker,
  l2.sid blocked,
  o.object_name
FROM v$lock l1
JOIN v$lock l2
  ON l1.id1 = l2.id1
JOIN dba_objects o
  ON l1.id1 = o.object_id
WHERE l1.block = 1;
Blocking Sessions
